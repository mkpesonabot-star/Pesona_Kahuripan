# Proyek Website Pesona Kahuripan

Website ini adalah Landing Page untuk penjualan perumahan subsidi **Pesona Kahuripan**.

## Rencana Pengembangan:
1. **Fase 1 (Selesai):** Pembuatan Landing Page dasar (Hero, Keunggulan, CTA WhatsApp).
2. **Fase 2:** Penambahan Galeri Foto (Interior & Eksterior), Denah Rumah, dan Spesifikasi Bangunan.
3. **Fase 3:** Integrasi Form Pendaftaran/Survey ke Google Sheets (menggunakan 'gog').
4. **Fase 4:** Optimasi SEO agar muncul di pencarian Google.
5. **Fase 5:** Pemasangan Google Ads untuk mendatangkan traffic.

## Folder Proyek:
- `index.html`: File utama website.
- `package.json`: Metadata proyek.
